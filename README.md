# Ada Lovelace

## Sobre o projeto
Esta landing page foi desenvolvida utilizando HTML, CSS e JavaScript com o objetivo de apresentar a história de Ada Lovelace.

## Desenvolvimento
A página foi estruturada com HTML, estilizada com CSS e possui um botão interativo feito com JavaScript.

## JavaScript
Foi utilizado um botão que mostra e esconde uma curiosidade ao ser clicado.

## Organização
- index.html → estrutura
- style.css → design
- script.js → interação
- imagens → fotos utilizadas
